#syscrawl

1. Install bower
```sh
  $ npm install -g bower
```
2. Install NuGet (through mono)
```sh
  $ brew install mono
```
3. Grant permissions to run the install script to your user
```sh
  $ chmod u+rwx configs/install.sh
```
4. And finally run the install script
```sh
  $ ./configs/install.sh
```

5. After, sync monodevelop project in unity
